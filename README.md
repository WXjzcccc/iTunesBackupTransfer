# iTunesBackupTransfer
iTunes备份解析
## Usage
```PlainText
~/iTunesBackupTransfer>python3 main.py -h
usage: main.py [-h] backup_path target_path

iTunesBackupTransfer

positional arguments:
  backup_path  iTunes备份目录
  target_path  备份解析目录

optional arguments:
  -h, --help   show this help message and exit
```
